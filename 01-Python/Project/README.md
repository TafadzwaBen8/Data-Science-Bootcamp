# 📊 Student Performance Analyzer

## 📌 Overview

The Student Performance Analyzer is a Python fundamentals project that allows users to enter student information, input marks for different subjects, calculate the total and average score, determine the student's grade, and display the final result.

This project was built as part of my Data Science learning journey through Jose Portilla's **Python for Data Science and Machine Learning Bootcamp**.

---

## 🚀 Features

- Enter student details
- Input marks for four subjects
- Validate marks between 0 and 100
- Calculate total marks
- Calculate average score
- Assign grades automatically
- Display pass/fail status
